# Azentrix Fullstack Task 1

## 📝 Description
This is a full-stack web application developed for Azentrix Task 1. It utilizes modern web technologies to deliver a robust and responsive user experience, featuring interactive data visualizations.

## 🚀 Technologies Used
Based on the project dependencies, this application utilizes:
* **Frontend:** React.js
* **Routing:** React Router (`react-router-dom`)
* **Data Visualization:** Recharts
* **Backend:** Node.js (Assumed for a full-stack environment)

## 🛠️ Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. Clone the repository (if you haven't already):
   ```bash
   git clone <your-github-repo-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd azentrix-fullstack-task1
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

### Running the Application
To start the development server, run the following command in your terminal:
```bash
npm start
```
*The application should now be running locally.*

* `src/` - Contains the React frontend code, components, and Redux store slices.
* `server/` - (If applicable) Contains the Node.js backend API.
